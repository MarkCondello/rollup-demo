Roll up

Minifying files can be done using terser plugin and running rollup in the terminal with this flag : --environment NODE_ENV:development

Node packages can be included using node-resolve & plugin-commonjs.

Providing support for es5 can be acheived using the getBabelOutputPlugin

Compiling scss is done through importing a main.scss file into the entry js file and using a plugin

Compilling Vue.js files is done with few configuations and a rollup-plugin-vue module.
